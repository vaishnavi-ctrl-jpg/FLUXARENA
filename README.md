## 🧠 Overview

**FluxArena** is a real-time **stadium intelligence dashboard** designed to improve physical event experiences at large sporting venues.

It helps attendees:

- Avoid crowd congestion  
- Reduce wait times  
- Find optimal routes  
- Navigate dynamically during crowd surges  
- Receive context-aware AI recommendations  

Built as a **submission-ready MVP** using **Next.js**, FluxArena combines **crowd simulation**, **AI decision logic**, and **premium mobile-first design** into a single intelligent system.

---

## ❗ Problem

Large-scale sporting venues often suffer from:

- Congested pathways  
- Unpredictable crowd surges  
- Long queues at gates and food zones  
- Inefficient attendee movement  
- Lack of real-time navigation support  

Traditional static maps fail in dynamic environments.

---

## 💡 Solution

FluxArena introduces a real-time intelligence layer that combines:

- Dynamic crowd heatmaps  
- Smart route optimization  
- AI-assisted recommendations  
- Surge simulation logic  
- Premium mobile-first interaction design

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

---

### Installation

Clone the repository:

```bash
git clone https://github.com/vaishnavi-ctrl-jpg/FluxArena.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

📱 Use a mobile viewport for the intended experience.
